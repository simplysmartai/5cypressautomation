Regression: Actual vs Predicted Rainfall Rainy Days Only (300-sample)

XGBoost Regression — Scatter Plot

Each point is one rainy day in 2019. X-axis = actual rainfall (inches). Y-axis = model prediction. Perfect predictions would hug the diagonal. Regression on rainfall amount is inherently difficult (R²=−0.79 reflects high variance in daily rain totals), but the model captures the ≈0.5" mean well.

Why negative R²? Rainfall amount on any given day is highly stochastic — even meteorologists struggle to predict exact totals. A negative R² means the model performs worse than simply predicting the mean, which is expected for daily precipitation regression without numerical weather model inputs. The classification task (will it rain?) is far more tractable and shows strong AUC > 0.69 across all models.
