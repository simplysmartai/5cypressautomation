Security & Data Protection

LAST UPDATED: FEBRUARY 19, 2026 // VERSION 1.0

1. Our Commitment

At 5 Cypress Automation, security is built into everything we do. We handle access to your business tools and workflows with the same care we'd expect for our own systems. This page documents how we protect your data and your business.

2. Data Transmission

All communication between your browser and our servers is encrypted using TLS 1.2 or higher (HTTPS). We enforce HSTS to prevent downgrade attacks. Form submissions, API calls, and file transfers are encrypted end-to-end.

3. Credential Handling

  • We never store API keys or passwords in plain text
  • Client credentials are stored in environment variables, never in source code or shared repositories
  • OAuth tokens (QuickBooks, Zoho, etc.) follow each provider's recommended secure flow
  • Access is scoped to the minimum permissions required for each automation

4. Access Controls

We apply least-privilege access principles across all integrations:

  • Each automation uses only the permissions it needs—nothing more
  • Admin tools and internal dashboards are protected by authentication
  • All API calls are logged for audit purposes
  • We revoke access immediately upon project completion or client request

5. Third-Party Services

Our automations connect to industry-standard platforms including QuickBooks Online, ShipStation, Stripe, Zoho, and Calendly. Each of these providers maintains their own security certifications (PCI DSS, SOC 2, etc.). We rely on their official APIs and do not store data from these platforms beyond what is necessary to complete a workflow.

6. Data Retention

  • We do not retain client data after a project ends unless explicitly agreed to for ongoing support
  • Logs are retained for 30 days for debugging purposes, then purged
  • You can request deletion of any data we hold at any time

7. Vulnerability Disclosure

If you discover a security issue in our systems, please report it responsibly before public disclosure. Contact us at nick@5cypress.com with details. We commit to acknowledging reports within 48 hours and resolving critical issues within 7 days.

8. Incident Response

In the event of a security incident affecting your data, we will notify you within 72 hours, provide a clear explanation of what happened, what data was affected, and what steps we are taking to resolve it and prevent recurrence.

9. Questions

Security concerns or questions? Reach us directly at: nick@5cypress.com
