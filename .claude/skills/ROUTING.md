# Skill Routing Guide

Use this guide when two skills look like they might overlap.

---

## Email

Use **`email-campaign`** — it's the only email skill.
Covers: B2B nurture sequences, re-engagement, cold outreach, announcements. Output is client-ready copy.

---

## Social Media

Use **`social-media-content`** — it's the only social skill.
Covers: LinkedIn posts, content calendars, thought leadership. Applies 40/30/20/10 framework.

---

## Backend Development

| Skill | When to Use |
|-------|-------------|
| `api-scaffolding` | Quickly spin up a new FastAPI or Express endpoint |
| `backend-development` | Architecture decisions, QBO/ShipStation integration patterns |
| `python-development` | Python scripts, FastAPI, async patterns, Pydantic, execution scripts |

**Rule:** New endpoint fast → `api-scaffolding`. Integration architecture → `backend-development`. Automation script → `python-development`.

---

## CRO (Conversion Rate Optimization)

| Skill | When to Use |
|-------|-------------|
| `page-cro` | Any public marketing page (homepage, landing page, pricing, feature pages) |
| `form-cro` | Non-signup forms — lead capture, contact, demo request |
| `popup-cro` | Exit-intent popups, modals, overlays, banners |
| `signup-flow-cro` | Account creation / registration / trial activation |
| `onboarding-cro` | Post-signup activation, first-run experience |
| `paywall-upgrade-cro` | In-app paywalls, upgrade prompts, feature gates |

## SEO

| Skill | When to Use |
|-------|-------------|
| `seo-audit` | Audit an existing site — diagnose problems, on-page issues |
| `programmatic-seo` | Build template-driven pages at scale for keyword targeting |
| `schema-markup` | Add JSON-LD / rich snippets to existing pages |

## Marketing Strategy & Content

| Skill | When to Use |
|-------|-------------|
