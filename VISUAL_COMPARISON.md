## üé≠ Atmospheric Effects

### 1. Film Grain Overlay
```css
/* Adds texture and depth */
opacity: 0.04;
mix-blend-mode: overlay;
```
Makes the site feel **tactile** and **premium** instead of flat digital.

### 2. Gradient Mesh
```css
/* Multi-point radial gradients */
radial-gradient(at 0% 0%, rgba(93, 140, 93, 0.15), transparent),
radial-gradient(at 100% 0%, rgba(251, 191, 36, 0.15), transparent)
```
Creates **atmospheric depth** without being obvious.

### 3. Mouse Trail Glow
```javascript
// Subtle 300px radial glow follows cursor
background: radial-gradient(circle, rgba(93, 140, 93, 0.08), transparent);
```
Adds **organic movement** and user connection.
